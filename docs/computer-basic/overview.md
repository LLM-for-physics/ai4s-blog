# 概述

本章将引导您掌握课程与实践项目中所必备的基础计算技能。内容涵盖 SSH 远程连接、VPN与网络代理配置、Python 编程基础、Git 版本控制，以及大语言模型（LLM）API 的调用方法。熟练运用这些工具将是您顺利完成本课程学习的关键。
