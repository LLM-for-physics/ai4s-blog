# 大作业：ai for science 应用探索

**发布时间**: 12月10日  
**截止时间**: 1月12日 23:59  
**权重**: 20%

## 📋 作业概述

本次大作业要求同学们在科研学习生活中探索 AI Agent 的可能应用场景，自主选题并完成一个具有实际意义的 AI 应用项目。

## 🎯 选题方向

主题自由，只要能够体现 AI 在实际场景中的应用价值即可。例如：收集某个领域的实验数据后用神经网络或其他机器学习方法去拟合数据，分析其中的特征和规律；把一个开源软件包封装为 MCP Server 供 LLM Agent 调用；开发一个 LLM Agent 自动阅读和解析 arXiv 论文并自动整理和解读其中的图片和数据表；开发一个 Code Agent 自动读论文并写代码复现论文中的 Pipeline；用 AI 解决自己生活中的其他问题减轻不必要的劳动……

## 📝 作业要求

### 提交内容

1. **项目名称**：为你的项目起一个简洁有意义的名字
2. **程序代码**：完整的项目代码，包括依赖配置和运行说明
3. **README 报告**：项目说明文档，介绍项目背景、解决方案、使用方法和结果展示等

### 提交位置

将所有代码和 README.md 文件放在 `~/assignments/assignment6/` 目录下。也可以将项目托管到 GitHub 并在 README 中附上仓库链接（不强制要求）。
